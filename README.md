# Manual SQA Audit & Security Analysis
### Project Overview
A comprehensive manual quality audit of the **Automation Exercise** e-commerce platform. This project demonstrates 7th-semester engineering rigor by combining functional SQA with network security analysis.

### Key Features Tested
* **User Authentication:** Robustness of signup/login flows.
* **Order Lifecycle:** Verified via **State Machine Diagrams** to ensure logical consistency.
* **Security Audit:**  Performed **XSS Injection** testing on search modules (Validated Input Sanitization).
    * Network traffic analysis using **Wireshark** to verify HTTPS/TLS data encryption.
* **Boundary Value Analysis:** Validated cart quantities and payment fields.

### Artifacts Included
* **Test Strategy:** Risk-based testing approach.
* **Test Case Suite:** Detailed test cases (Functional, Negative, and Security).
* **Defect Log:** Detailed bug reports with severity and priority.
* **Logic Modeling:** State Machine Diagram for the Checkout flow.
